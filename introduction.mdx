`brew-cask-upgrade` (`brew cu`) is a command-line tool for upgrading every outdated app installed by [Homebrew Cask](https://github.com/Homebrew/homebrew-cask). It replaces the native `brew upgrade --cask` command with a smarter upgrade experience: an interactive interface, version pinning, glob pattern targeting, and awareness of apps that self-update.

## Why brew-cask-upgrade exists

Homebrew Cask manages the installation of macOS GUI applications and large binaries. Its built-in `brew upgrade --cask` upgrades everything at once, with no prompting and no way to exclude specific apps.

`brew-cask-upgrade` fills that gap. It shows you a table of outdated apps before doing anything, lets you confirm or skip each one, and gives you tools to permanently exclude apps you don't want auto-upgraded. It is designed for users who want upgrade control without giving up the convenience of Homebrew.

## How it relates to Homebrew Cask

`brew-cask-upgrade` is an **external Homebrew command** — a plugin that Homebrew loads when you tap the `buo/cask-upgrade` repository. Once tapped, it is available as `brew cu` alongside your existing Homebrew commands. It reads from and writes to the same Cask installation that Homebrew manages; it does not maintain a separate package database.

Because it is external, it must be installed separately:

```bash
brew tap buo/cask-upgrade
```

After that, `brew cu` is available system-wide. See [Installation](/installation) for full setup steps.

## The brew cu command

`brew cu` is the entry point for everything in this tool. The full syntax is:

```bash
brew cu [command] [CASK] [options]
```

The default command (`run`) checks for outdated casks and starts the upgrade flow. Additional subcommands manage version pinning:

| Command | Description |
|---------|-------------|
| `brew cu` | Upgrade all outdated casks (prompts before upgrading) |
| `brew cu CASK` | Upgrade a specific cask by name or glob pattern |
| `brew cu pin CASK` | Pin a cask at its current version |
| `brew cu unpin CASK` | Remove a pin from a cask |
| `brew cu pinned` | List all currently pinned casks |
| `brew cu livecheck` | Run `brew livecheck` on all installed casks |

Run `brew help cu` at any time to display the full usage guide in your terminal.

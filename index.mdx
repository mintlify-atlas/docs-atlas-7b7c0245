`brew-cask-upgrade` (`brew cu`) is a command-line tool that replaces the native `brew upgrade --cask` command with a smarter, more interactive upgrade experience. It detects outdated Homebrew Cask apps, shows you what's available, and lets you upgrade them individually or all at once.

## What brew-cask-upgrade does

<CardGroup cols={2}>
  <Card title="Smart upgrade detection" icon="magnifying-glass">
    Finds all outdated Homebrew Cask apps and shows their current and available versions before upgrading.
  </Card>
  <Card title="Interactive mode" icon="computer-mouse">
    Step through each outdated app one by one — upgrade, skip, or pin each one individually.
  </Card>
  <Card title="Version pinning" icon="lock">
    Exclude specific apps from upgrades until you're ready. Pin, unpin, and export your pin list.
  </Card>
  <Card title="Glob pattern support" icon="asterisk">
    Upgrade a subset of apps matching a pattern, like `brew cu 'firefox*'` or `brew cu 'adobe-*'`.
  </Card>
  <Card title="Auto-update awareness" icon="rotate">
    Skips apps that self-update by default. Use `--all` to include them when you want.
  </Card>
  <Card title="Mac App Store integration" icon="apple">
    Optionally include Mac App Store apps via `mas-cli` with `--include-mas`.
  </Card>
</CardGroup>

## Get started

<Steps>
  <Step title="Tap the repository">
    ```bash
    brew tap buo/cask-upgrade
    ```
  </Step>
  <Step title="Run your first upgrade">
    ```bash
    brew cu
    ```
    This checks for updates, shows you what's outdated, and prompts you to upgrade.
  </Step>
  <Step title="Explore options">
    Upgrade all apps including auto-updaters, use interactive mode, or pin specific apps. See the [options reference](/configuration/options-reference) for the full list.
  </Step>
</Steps>

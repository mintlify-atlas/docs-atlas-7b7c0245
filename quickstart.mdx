---
title: "Quick start"
description: "Get brew-cask-upgrade installed and run your first cask upgrade in under a minute."
---

<Steps>
  <Step title="Install brew cu">
    Tap the repository to make the `brew cu` command available:

    ```bash
    brew tap buo/cask-upgrade
    ```
  </Step>

  <Step title="Run brew cu">
    Start the upgrade flow for all outdated casks:

    ```bash
    brew cu
    ```

    `brew cu` first runs `brew update` to fetch the latest cask definitions, then scans your installed apps and compares versions. When it finds outdated apps, it prints a table like this:

    ```
    ==> Updating Homebrew
    Already up-to-date.

    ==> Finding outdated apps

    Cask                        Current          Latest           Result
    -------------------------   --------------   --------------   --------
    firefox                     120.0            121.0            outdated
    visual-studio-code          1.84.2           1.85.1           outdated
    spotify                     1.2.25.1011      1.2.26.1187      PASS
    rectangle                   0.64             0.65             outdated

    ==> 3 outdated apps (excluding 1 with auto-update)

    Update all outdated apps? [y/i/N]
    ```

    Apps marked `PASS` have built-in auto-update and are skipped by default. Use `--all` to include them.
  </Step>

  <Step title="Respond to the prompt">
    At the `[y/i/N]` prompt you have three choices:

    | Input | Action |
    |-------|--------|
    | `y` | Upgrade all outdated apps immediately |
    | `i` | Enter interactive mode — confirm each app one by one |
    | `N` | Cancel without upgrading anything |

    In **interactive mode**, you are prompted for each app separately:

    ```
    Do you want to upgrade "firefox", [p]in it to exclude it from updates or [q]uit [y/p/q/N]?
    ```

    - `y` — upgrade this app
    - `p` — pin this app at its current version (skips it in all future runs)
    - `q` — quit interactive mode entirely
    - `N` — skip this app for now
  </Step>

  <Step title="Upgrade all without prompting">
    To skip the confirmation prompt and upgrade everything automatically, pass the `-y` flag:

    ```bash
    brew cu -y
    ```

    <Tip>
      Use `brew cu -y` in scheduled tasks or when you want a fully unattended upgrade run. Combine it with `--cleanup` to also clear the download cache after upgrading: `brew cu -y --cleanup`.
    </Tip>
  </Step>

  <Step title="Upgrade a specific app">
    Pass a cask name to upgrade just that app:

    ```bash
    brew cu firefox
    ```

    Expected output:

    ```
    ==> Updating Homebrew
    Already up-to-date.

    ==> Finding outdated apps

    Cask         Current    Latest     Result
    ----------   --------   --------   --------
    firefox      120.0      121.0      outdated

    ==> 1 outdated app

    Update all outdated apps? [y/i/N] y

    ==> Upgrading firefox
    ==> Downloading https://releases.mozilla.org/pub/firefox/releases/121.0/mac/...
    ######################################################################## 100.0%
    ==> Upgrading App 'Firefox.app'
    firefox was successfully upgraded!
    ```

    You can also use glob patterns to target multiple related apps at once. Always quote the pattern to prevent your shell from expanding it before `brew cu` sees it:

    ```bash
    brew cu 'adobe-*'
    ```
  </Step>
</Steps>

For more detail on all available flags and options, see [Upgrading apps](/usage/upgrading-apps).
